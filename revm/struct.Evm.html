Struct Evm

Source 
pub struct Evm<'a, EXT, DB: Database> {
    pub context: Context<EXT, DB>,
    pub handler: Handler<'a, Context<EXT, DB>, EXT, DB>,
}
Expand description

EVM instance containing both internal EVM context and external context and the handler that dictates the logic of EVM (or hardfork specification).

Fields§

§context: Context<EXT, DB>

Context of execution, containing both EVM and external context.

§handler: Handler<'a, Context<EXT, DB>, EXT, DB>

Handler is a component of the of EVM that contains all the logic. Handler contains specification id and it different depending on the specified fork.
